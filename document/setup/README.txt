		   ===================
		   Tài liệu Người dùng
		   ===================
		     PHẦN MỀM EDUSYS
		     ===============


Hướng dẫn cài đặt triển khai
----------------------------

• Cài SQL Server 2008 trở lên
• Tạo CSDL EduSys bằng cách chạy file EduSys.sql
• Cài đặt JDK 1.8 trở lên
• Chạy file edusys_setup.exe



Hướng dẫn sử dụng phần mềm
--------------------------

• Đăng nhập với tài khoản username/password: admin/admin
• Cấp các tài khoản có vai trò là trưởng phòng
• Đăng nhập tài khoản mới và xóa tài khoản admin
• Xem hướng dẫn sử dụng trong menu trợ giúp của phần mềm
